# h2p1 Top K Frequent Elements

LINK - https://neetcode.io/problems/top-k-elements-in-list/question?list=neetcode150

```python
class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        return [num for num,freq in Counter(nums).most_common(k)]
        return [num for num,freq in Counter(nums).most_]
```
